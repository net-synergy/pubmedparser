Contains scripts for converting Pubmed xml files to node and relationship csv files then importing them into neo4j. See [[https://dtd.nlm.nih.gov/ncbi/pubmed/doc/out/190101/index.html][pubmed_190101.dtd]] for information on xml files.

Files ~nodes.list~ and ~relationships.list~ contain the element names in the xml files that should be collected in csv files as nodes and relationships.
